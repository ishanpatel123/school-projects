How to read the window.config file
----------------------------------------------
1.	The text file located as Pedogoge\BasicWindow\window.config.txt

2.	File Format is very simple to change any values in the file. All the labels are stored on the left side of the line, and all values are stored in the right side of line after ‘: ’ word. Also, make sure there is a space between character ‘:’ and value.
	For example, change the value from:
		Title: Hello World!
		initial position x: 200
		background red: 0.2f
	To:
		Title: How are you?
		initial position x: 600
		background red: 0.7f
